# REALMS — Deutsche Fassung

Die deutsche Ausgabe des REALMS-Manuskripts liegt in diesem Ordner. PDF aus den Markdown-Dateien erzeugen:

```bash
./scripts/build-pdf-de.sh              # → dist/manuscript-de.pdf
./scripts/build-pdf-theoretische.sh    # → dist/theoretische-grundlagen.pdf
```

(aus dem Repository-Root ausführen)
